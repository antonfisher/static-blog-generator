# Static Site Generator

Uses for: [http://antonfisher.github.io](http://antonfisher.github.io)

## Features
* Static HTML generation (Index, Post, About pages)
* Markdown
* Nunjucks Templates for theming
* Gulp based
* RSS feed generator
